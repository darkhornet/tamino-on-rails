find(*arguments)

Discovers object(s) in the Tamino database.

The first argument can either be one of the following:

:all:finds all matching entries and returns an Array of objects representing them
:first:finds the first matching entry and returns an object representing it
a primary key value:finds an entry by its primary key value (denoted as ino:id attribute)

When used with :all or :first, :condition as second argument will limit results. It has to be an X-Query passed as a string value.

Example: 

 Person.find(:all, :condition => "name/firstName=\"John\"") # finds all Johns
 Person.find(42) # find element with an ino:id attribute value of 42
